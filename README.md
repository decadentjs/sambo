# sambo

Converts a list of strings into a compact, readable representation with delimiters of choice

```
sambo alba,albero,albergo,alberto
# => alb(a|er(go|o|to))

sambo aria:arianna : [ + ]
# => aria[+nna]
```

# why?

Useful when you need to pass long lists of URL params
